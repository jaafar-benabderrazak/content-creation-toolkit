Create shared/branding_clips.py. No new pip dependencies — uses only FFmpeg subprocess and stdlib.

Design: Each clip is a 3-second (intro) or 4-second (outro) 1920x1080 black-background video with:
- Channel name in white centered text (fontsize 72, bold if font available)
- Tagline text below channel name (fontsize 36, gray #aaaaaa)
- Avatar image overlaid centered-top if avatar file exists (scaled to 160x160, positioned at x=(W-160)/2, y=80)
- Fade-in for intro (0 to 1s), fade-out for outro (last 1s)

Implementation:

1. _run_ffmpeg(args, description) — mirror of post_process.py version (copy exactly, do not import):
   ```python
   def _run_ffmpeg(args: list[str], description: str) -> None:
       import subprocess
       cmd = ["ffmpeg", "-y"] + args
       logger.info(f"[BrandingClip] {description}")
       result = subprocess.run(cmd, capture_output=True, text=True)
       if result.returncode != 0:
           raise RuntimeError(f"FFmpeg failed during '{description}': {result.stderr[:500]}")
   ```

2. _escape_drawtext(text: str) -> str:
   - Escape single quotes, backslashes, and colons for FFmpeg drawtext filter
   - text.replace("\\", "\\\\").replace("'", "\\'").replace(":", "\\:")

3. _find_font() -> str:
   - Try paths: "C:/Windows/Fonts/arialbd.ttf", "C:/Windows/Fonts/Arial.ttf", "/usr/share/fonts/truetype/dejavu/DejaVuSans-Bold.ttf"
   - Return path if exists, else return "" (FFmpeg uses default font when fontfile empty)

4. generate_intro_clip(branding: BrandingData, output_path: Path, duration: float = 3.0) -> Path:
   - Build FFmpeg filter_complex:
     a. Base: black background lavfi color=black:s=1920x1080:d={duration}
     b. Channel name: drawtext centered at y=h/2+40 (below avatar), fontsize=72, fontcolor=white, fade=in:0:30 (1s at 30fps)
     c. Tagline: drawtext centered at y=h/2+130, fontsize=36, fontcolor=#aaaaaa
     d. If branding.avatar_local_path exists and Path(branding.avatar_local_path).exists():
        - Add -i {avatar_path} input, scale=160:160, overlay at x=(W-160)/2:y=80
   - Use separate avatar-present vs avatar-absent FFmpeg command builds (simpler than dynamic filter_complex assembly):

     WITHOUT avatar:
     ```
     ffmpeg -y -f lavfi -i color=black:s=1920x1080:d={duration}:r=30
       -vf "drawtext=fontfile='{font}':text='{name}':fontsize=72:fontcolor=white:x=(w-tw)/2:y=(h-th)/2+40,
            drawtext=fontfile='{font}':text='{tagline}':fontsize=36:fontcolor=#aaaaaa:x=(w-tw)/2:y=(h-th)/2+130,
            fade=in:0:30"
       -an {output}
     ```

     WITH avatar (use filter_complex):
     ```
     ffmpeg -y -f lavfi -i color=black:s=1920x1080:d={duration}:r=30 -i {avatar}
       -filter_complex "[1:v]scale=160:160[logo];[0:v][logo]overlay=x=(W-160)/2:y=80,
            drawtext=fontfile='{font}':text='{name}':fontsize=72:fontcolor=white:x=(w-tw)/2:y=(h-th)/2+40,
            drawtext=fontfile='{font}':text='{tagline}':fontsize=36:fontcolor=#aaaaaa:x=(w-tw)/2:y=(h-th)/2+130,
            fade=in:0:30[v]" -map "[v]" -an {output}
     ```

   - output_path.parent.mkdir(parents=True, exist_ok=True)
   - Return output_path

5. generate_outro_clip(branding: BrandingData, output_path: Path, duration: float = 4.0) -> Path:
   - Same structure as intro but fade=out:st={duration-1}:d=30 (last 1s fade-out)
   - Tagline text: "Thanks for watching!" if tagline is empty or is just the channel_name, else use branding.tagline
   - Return output_path

6. generate_branding_clips(branding: BrandingData, cache_dir: Path) -> tuple[Path, Path]:
   - intro_path = cache_dir / "intro.mp4"
   - outro_path = cache_dir / "outro.mp4"
   - Generate both; return (intro_path, outro_path)
   - Regenerate if files do not exist (callers control when to call this)

Module-level imports: logging, pathlib.Path only. All subprocess work inside functions.
