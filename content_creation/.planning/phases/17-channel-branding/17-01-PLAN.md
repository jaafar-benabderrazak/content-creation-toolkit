<task type="auto">
  <name>Task 1: Create shared/branding.py with channel fetch, avatar download, and cache</name>
  <files>shared/branding.py</files>
  <action>
Create shared/branding.py with the following:

1. BrandingData dataclass (or TypedDict):
   - channel_name: str
   - avatar_url: str
   - avatar_local_path: str  (absolute path to .cache/branding/avatar.jpg)
   - description: str
   - tagline: str  (first sentence of description, max 80 chars, stripped)
   - fetched_at: str  (ISO datetime)

2. Constants:
   - CACHE_DIR = Path(".cache/branding")
   - CACHE_FILE = CACHE_DIR / "branding.json"
   - AVATAR_FILE = CACHE_DIR / "avatar.jpg"
   - CACHE_TTL_HOURS = 24 * 7  (7 days — explicit refresh is the primary invalidation mechanism)

3. _load_cache() -> Optional[BrandingData]:
   - Returns None if CACHE_FILE does not exist
   - Returns None if fetched_at is older than CACHE_TTL_HOURS
   - Returns BrandingData from JSON otherwise
   - Catches json.JSONDecodeError and returns None (corrupt cache = re-fetch)

4. _save_cache(data: BrandingData) -> None:
   - CACHE_DIR.mkdir(parents=True, exist_ok=True)
   - Writes CACHE_FILE as JSON (dataclasses.asdict or dict)

5. _download_avatar(url: str) -> Path:
   - Downloads avatar image to AVATAR_FILE using urllib.request.urlretrieve
   - Returns AVATAR_FILE path
   - Logs warning and returns AVATAR_FILE (even if missing) on failure

6. _get_youtube_service():
   - Mirrors publisher.py pattern exactly: try TOKEN_FILE (json), then TOKEN_PICKLE (pickle)
   - SCOPES = ["https://www.googleapis.com/auth/youtube.upload"]  (upload scope covers channels.list)
   - Uses google.oauth2.credentials.Credentials + google-api-python-client build()
   - Raises ImportError with pip install hint if google libs missing
   - Raises FileNotFoundError if neither token file exists

7. fetch_channel_branding(refresh: bool = False) -> BrandingData:
   - If not refresh: check _load_cache(); return cached if valid
   - If refresh: delete CACHE_FILE and AVATAR_FILE if they exist (missing_ok=True)
   - Call _get_youtube_service(), then service.channels().list(part="snippet", mine=True, maxResults=1).execute()
   - Extract items[0].snippet: title -> channel_name, description -> description, thumbnails.high.url -> avatar_url
   - If items is empty: raise RuntimeError("No YouTube channel found for authenticated account")
   - tagline = first sentence of description up to 80 chars; if description empty, use channel_name
   - _download_avatar(avatar_url)
   - Construct BrandingData, call _save_cache(), return it
   - Log [Branding] fetched channel: {channel_name} on fetch, [Branding] loaded from cache on cache hit

Import guard pattern (same as publisher.py): wrap google imports inside function bodies so import fails gracefully at module load; raise ImportError with install hint at call time.
