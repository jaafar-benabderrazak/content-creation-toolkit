Create `generators/sdxl.py` implementing the `SDXLGenerator` class. Full specification:

**Imports:**
```python
from __future__ import annotations
import hashlib, json, logging, random
from pathlib import Path
from typing import Any, Dict, List, Optional, Tuple
from config.prompt_template import PromptTemplate, build_compel_prompt
from config.pipeline_config import SDXLSettings
```

**Module-level constants** (extracted from existing study_with_me_generator.py):
- `STYLE_VARIATIONS`: list of style strings (copy verbatim from existing study_with_me_generator.py `STYLE_VARIATIONS` list)
- `WEATHER_EFFECTS`: list of weather strings (copy verbatim from existing `WEATHER_EFFECTS` list)
- `MODEL_ID = "stabilityai/stable-diffusion-xl-base-1.0"`
- `CACHE_DIR_NAME = ".cache/images"`

**`_cache_key(params: dict) -> str` (module-level private function):**
- `json.dumps(params, sort_keys=True).encode()` → `hashlib.sha256(...).hexdigest()`
- Returns full 64-char hex digest (NOT truncated — SDXL-02 requires full-parameter uniqueness)

**`SDXLGenerator` class:**

`__init__(self, cache_dir: Path | None = None)`:
- Sets `self.cache_dir = Path(cache_dir) if cache_dir else Path(CACHE_DIR_NAME)`
- Creates `self.cache_dir` if it doesn't exist: `self.cache_dir.mkdir(parents=True, exist_ok=True)`
- Sets `self.logger = logging.getLogger(__name__)`

`build_prompt(self, scene: dict, sdxl_cfg: SDXLSettings, extra_context: dict | None = None) -> Tuple[str, str]`:
- `context = {**scene, **(extra_context or {})}`
- For positive: call `PromptTemplate.render(scene["prompt"], context, quality_suffix=sdxl_cfg.quality_suffix)` — if scene["prompt"] contains no `{variables}`, render() still works (no substitution needed)
- Append a random style variation: `style = random.choice(STYLE_VARIATIONS)`; append to resolved prompt: `f"{resolved}, {style}"`
- If `scene.get("environment") == "indoor"` and `sdxl_cfg` steps >= 25 (medium/high): append a random weather effect
- Negative: return `sdxl_cfg.negative_prompt` unchanged
- Return `(positive_prompt, negative_prompt)`

`_params_dict(self, positive: str, negative: str, sdxl_cfg: SDXLSettings, profile: str, seed: int) -> dict`:
- Return `{"prompt": positive, "negative_prompt": negative, "steps": sdxl_cfg.steps, "guidance_scale": sdxl_cfg.guidance_scale, "width": sdxl_cfg.width, "height": sdxl_cfg.height, "profile": profile, "seed": seed, "model_id": MODEL_ID}`

`_sidecar_path(self, image_path: Path) -> Path`:
- Return `image_path.with_suffix(".json")`

`_write_sidecar(self, image_path: Path, params: dict) -> None`:
- Write `json.dumps(params, indent=2)` to `self._sidecar_path(image_path)` with UTF-8 encoding

`_cache_lookup(self, key: str) -> Optional[Path]`:
- glob `self.cache_dir` for `f"*_{key}.png"` — return first match if found and sidecar JSON also exists, else return None

`generate_batch(self, scenes: List[dict], sdxl_cfg: SDXLSettings, profile: str, out_dir: Path, seed_offset: int = 42, force: bool = False) -> List[Path]`:
- Iterates scenes with index `i`
- For each scene, calls `build_prompt(scene, sdxl_cfg)` → `(positive, negative)`
- Computes `seed = seed_offset + i`
- Builds `params = self._params_dict(positive, negative, sdxl_cfg, profile, seed)`
- Computes `key = _cache_key(params)`
- If not force: calls `self._cache_lookup(key)` — if hit: `self.logger.info(f"[SDXL] Cache hit: scene {i} (key={key[:8]})")`, appends cached path, continue
- On miss: `self.logger.info(f"[SDXL] Generating scene {i} ...")`
- Calls `_generate_one(positive, negative, sdxl_cfg, seed)` → PIL Image
- Saves to `out_dir / f"scene_{i:03d}_{key[:12]}.png"` at quality 95
- Calls `_write_sidecar(image_path, params)`
- Appends path, continues
- Returns list of paths

`_generate_one(self, positive: str, negative: str, sdxl_cfg: SDXLSettings, seed: int) -> Any`:
- All GPU/model imports inside this method (lazy import pattern to allow module import without GPU):
  ```python
  import torch
  from diffusers import StableDiffusionXLPipeline, DPMSolverMultistepScheduler
  ```
- Device detection: `torch.cuda.is_available()` → cuda/cpu, float16/float32
- `pipe = StableDiffusionXLPipeline.from_pretrained(MODEL_ID, torch_dtype=..., use_safetensors=True, variant="fp16" if cuda else None)`
- `pipe.scheduler = DPMSolverMultistepScheduler.from_config(pipe.scheduler.config)`
- `pipe = pipe.to(device)`
- On CUDA: `pipe.enable_model_cpu_offload()`, `pipe.enable_vae_slicing()`, `pipe.enable_attention_slicing(1)`
- Try `build_compel_prompt(positive, negative, pipe)` for compel conditioning; if `ImportError` falls back to plain string prompts
- On compel success: call `pipe(prompt_embeds=cond, pooled_prompt_embeds=pooled, negative_prompt_embeds=neg_cond, negative_pooled_prompt_embeds=neg_pooled, num_inference_steps=sdxl_cfg.steps, guidance_scale=sdxl_cfg.guidance_scale, width=sdxl_cfg.width, height=sdxl_cfg.height, generator=torch.manual_seed(seed)).images[0]`
- On compel fallback: call `pipe(prompt=positive, negative_prompt=negative, num_inference_steps=sdxl_cfg.steps, guidance_scale=sdxl_cfg.guidance_scale, width=sdxl_cfg.width, height=sdxl_cfg.height, generator=torch.manual_seed(seed)).images[0]`
- `del pipe; torch.cuda.empty_cache()` after generation
- Returns PIL Image

**Note:** `_generate_one` loads and unloads the model per batch call. This is intentional (matches existing pattern in study_with_me_generator.py) — model lifecycle management is not in scope for Phase 10.

**Do NOT import `PipelineConfig` in this file** — generators receive only their settings sub-model (`SDXLSettings`) per the architecture boundary.
