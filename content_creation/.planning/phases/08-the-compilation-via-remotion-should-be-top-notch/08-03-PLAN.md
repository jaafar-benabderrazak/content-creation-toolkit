    The existing title uses `interpolate(frame, [0, 30], [0, 1])` for opacity and `[30, 0]` for translateY. Replace with spring:
    ```tsx
    const titleSpring = spring({
      frame,
      fps,
      config: profileConfig.springConfig,
      durationInFrames: 25,
    });
    const titleOpacity = titleSpring;
    const titleY = interpolate(titleSpring, [0, 1], [30, 0]);
    ```
    Apply `fontFamily` to the title div style.

    **Bullets — replace per-bullet interpolate with TextReveal:**
    Remove the existing bullets.map() block that uses `interpolate(frame, [start, start+20], [0, 1])` for opacity and `[40, 0]` for translateX.

    Replace with TextReveal per bullet. Each bullet gets its own TextReveal with a startFrame offset:
    ```tsx
    <div style={{ display: "flex", flexDirection: "column", gap: 24 }}>
      {bullets.map((bullet, i) => {
        const bulletStart = 60 + i * 45; // same timing as original
        return (
          <div
            key={i}
            style={{
              fontSize: 42,
              color: "rgba(255, 255, 255, 0.95)",
              display: "flex",
              alignItems: "center",
              gap: 16,
              fontFamily,
            }}
          >
            <span style={{ color: "#60a5fa", fontSize: 36 }}>▸</span>
            <TextReveal
              text={bullet}
              startFrame={bulletStart}
              springConfig={profileConfig.springConfig}
              staggerFrames={4}
              wordStyle={{ fontSize: 42, color: "rgba(255, 255, 255, 0.95)" }}
            />
          </div>
        );
      })}
    </div>
    ```

    **Background image:** Apply `filter: profileConfig.cssColorFilter` to the background Img style.

    **End fade-out:** Replace the bare `interpolate(frame, [durationInFrames - 30, durationInFrames], [0, 1])` with eased version:
    ```tsx
    opacity: interpolate(
      frame,
      [durationInFrames - 30, durationInFrames],
      [0, 1],
      {
    ```
