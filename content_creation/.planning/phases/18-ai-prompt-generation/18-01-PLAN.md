PromptGenerationError(Exception): raised when OPENAI_API_KEY is absent, the API call raises any exception, or the JSON response is missing required keys.

PromptGenerator class:

  __init__(self, api_key: str | None = None, model: str = "gpt-4o-mini"):
    - api_key defaults to os.environ.get("OPENAI_API_KEY")
    - If api_key is None after env lookup, raise PromptGenerationError("OPENAI_API_KEY not set")
    - Instantiate openai.OpenAI(api_key=api_key) — use openai>=1.0 client (not legacy openai.ChatCompletion)

  generate(self, tags: str, profile_style: str) -> dict:
    - tags: comma-separated user input, e.g. "lofi, rain, cozy, study"
    - profile_style: profile's style string from cinematic.yaml video.style_prompt field, e.g. "cinematic wide shot, dramatic lighting, film photography"
    - Builds a system prompt:
        "You are a professional AI image prompt engineer. Generate prompts for a {profile_style} visual aesthetic. Return only valid JSON, no markdown."
    - Builds a user prompt:
        "Tags: {tags}\n\nGenerate:\n- positive_prompt: one SDXL positive prompt (50-80 words) that captures the tags in the {profile_style} aesthetic\n- negative_prompt: SDXL negative prompt (5-10 terms) optimized for this style\n- scene_templates: list of exactly 8 distinct scene variation prompts, each 40-60 words, covering different moments/lighting/compositions derived from the tags\n- music_prompt: Suno-compatible music prompt (20-30 words) matching the mood of the tags\n\nReturn JSON only."
    - Call self._client.chat.completions.create(model=self.model, messages=[system, user], temperature=0.8, response_format={"type": "json_object"})
    - Parse response: content = response.choices[0].message.content; data = json.loads(content)
    - Validate required keys: {"positive_prompt", "negative_prompt", "scene_templates", "music_prompt"}
    - Validate scene_templates is a list of exactly 8 strings — if fewer than 8, raise PromptGenerationError with count
    - Return data dict as-is (callers access keys directly)
    - Wrap entire API call + parse in try/except Exception as e: raise PromptGenerationError(str(e)) from e

  _build_profile_style(config: PipelineConfig) -> str (staticmethod):
    - Returns config.video.style_prompt
    - Convenience helper so callers don't need to know which field to read

Module-level imports needed: os, json, openai (import openai).
