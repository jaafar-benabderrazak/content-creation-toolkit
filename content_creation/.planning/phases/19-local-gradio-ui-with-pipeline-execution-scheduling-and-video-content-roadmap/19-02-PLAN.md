**RoadmapEntry dataclass:**
```python
@dataclass
class RoadmapEntry:
    id: str             # uuid4 hex
    title: str          # human-readable video title
    tags: str           # comma-separated content tags
    profile: str        # profile name (e.g. "cinematic")
    notes: str          # free-form notes
    status: str         # planned | producing | published
    created_at: str     # ISO 8601
    updated_at: str     # ISO 8601
```

**VideoRoadmap class:**
- `__init__(self, roadmap_file: Path = Path("video_roadmap.json"))` — loads existing entries from JSON; initializes `self._entries: List[RoadmapEntry]`.
- `add_entry(title, tags, profile, notes="") -> RoadmapEntry` — creates entry with uuid4 id, status="planned", timestamps now; appends; saves; returns entry.
- `update_status(entry_id: str, status: str) -> bool` — validates status in `{"planned", "producing", "published"}`, sets status and updated_at, saves. Returns True if found.
- `update_entry(entry_id: str, title: str = None, tags: str = None, profile: str = None, notes: str = None) -> bool` — updates any non-None fields and sets updated_at, saves. Returns True if found.
- `delete_entry(entry_id: str) -> bool` — removes entry by id, saves. Returns True if found.
- `move_up(entry_id: str) -> bool` — swaps entry with the one before it in the list. Returns True if swapped (False if already first).
- `move_down(entry_id: str) -> bool` — swaps entry with the one after it in the list. Returns True if swapped (False if already last).
- `list_entries(status_filter: str = None) -> List[RoadmapEntry]` — returns entries in order; filtered to `status_filter` if provided; returns a copy (not the internal list).
- `_save(self)` — atomically writes `[asdict(e) for e in self._entries]` to `self._roadmap_file` (write to `.tmp`, then `Path.replace`).
- `_load(self) -> List[RoadmapEntry]` — reads JSON, reconstructs `List[RoadmapEntry]`. Returns `[]` on FileNotFoundError or empty file.

**Module-level singleton (lazy):**
```python
_roadmap: Optional[VideoRoadmap] = None

def get_roadmap() -> VideoRoadmap:
    global _roadmap
    if _roadmap is None:
        _roadmap = VideoRoadmap()
    return _roadmap
```

import tempfile
from pathlib import Path
from roadmap import VideoRoadmap

tmp = Path(tempfile.mktemp(suffix=".json"))
r = VideoRoadmap(roadmap_file=tmp)

# Add
e1 = r.add_entry("Cozy Rain Study", "lofi, rain, cozy", "lofi_study", "First video")
e2 = r.add_entry("Epic Focus", "cinematic, focus", "cinematic", "")
assert len(r.list_entries()) == 2

# Status update
ok = r.update_status(e1.id, "producing")
assert ok
assert r.list_entries()[0].status == "producing"

# Edit
r.update_entry(e1.id, title="Cozy Rain Study — Updated")
assert r.list_entries()[0].title == "Cozy Rain Study — Updated"

# Move down / up
r.move_down(e1.id)
assert r.list_entries()[0].id == e2.id  # e2 now first
r.move_up(e2.id)
assert r.list_entries()[0].id == e2.id  # e2 already first; no change

# Filter
r.update_status(e2.id, "published")
planned = r.list_entries(status_filter="planned")
assert len(planned) == 0
published = r.list_entries(status_filter="published")
assert len(published) == 1

# Delete
r.delete_entry(e1.id)
assert len(r.list_entries()) == 1

# Persistence — reload from same file
r2 = VideoRoadmap(roadmap_file=tmp)
assert len(r2.list_entries()) == 1

print("PASS")
