**Job dataclass:**
```python
@dataclass
class Job:
    id: str                     # uuid4 hex
    profile: str                # profile name (e.g. "cinematic")
    tags: str                   # comma-separated tags or "" if not used
    output_path: str            # e.g. "out/video.mp4"
    scheduled_at: str           # ISO 8601 string
    status: str                 # pending | running | done | failed
    created_at: str             # ISO 8601 string
    log: str = ""               # last N lines of stdout captured
```

**JobQueue class:**
- `__init__(self, queue_file: Path = Path("jobs.json"))` — loads existing jobs from JSON; starts APScheduler BackgroundScheduler (daemon=True); reschedules all `pending` jobs that are in the future.
- `add_job(profile, tags, output_path, scheduled_at_iso) -> Job` — creates Job with uuid4 id, appends to `self._jobs`, calls `_schedule(job)`, saves to JSON. Returns the Job.
- `cancel_job(job_id: str) -> bool` — sets status to "failed" (cancelled), removes APScheduler job if exists, saves. Returns True if found.
- `list_jobs() -> List[Job]` — returns copy of `self._jobs` sorted by `scheduled_at` ascending.
- `_schedule(job: Job)` — adds APScheduler `DateTrigger` job targeting `_run_job(job.id)`. If `scheduled_at` is in the past, fires immediately via `_run_job` in a thread.
- `_run_job(job_id: str)` — finds job by id, sets status="running", saves, calls `_launch_pipeline(job)`, sets status="done" or "failed" based on returncode, saves.
- `_launch_pipeline(job: Job) -> int` — builds command: `[sys.executable, "study_with_me_generator.py", "--config", "configs/last_run.yaml", "--out", job.output_path]`; if `job.tags` is non-empty, appends `["--tags", job.tags]`; runs with `subprocess.Popen(cmd, stdout=subprocess.PIPE, stderr=subprocess.STDOUT, text=True)`; streams stdout lines into `job.log` (keep last 200 lines); returns process returncode.
- `_save(self)` — writes `[asdict(j) for j in self._jobs]` to `self._queue_file` atomically (write to `.tmp` then `Path.replace`).
- `_load(self)` — reads JSON, reconstructs `List[Job]` via `Job(**d)` for each entry. Returns `[]` on FileNotFoundError.

**Module-level singleton (lazy):**
```python
_queue: Optional[JobQueue] = None

def get_queue() -> JobQueue:
    global _queue
    if _queue is None:
        _queue = JobQueue()
    return _queue
```
