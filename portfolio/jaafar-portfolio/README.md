
# Jaafar Benabderrazak – Portfolio

A fast, modern portfolio built with Next.js 14 (App Router), Tailwind CSS, Lucide icons, and Framer Motion.

## Quick start
```bash
npm install
npm run dev
# open http://localhost:3000
```

Put your resume at `public/resume.pdf` (a placeholder is included).

## Deploy to Vercel
1. Push this repo to GitHub.
2. Create a new project on https://vercel.com and import the repo.
3. Use framework = Next.js. Default settings are fine.

## Notes
- Dark mode persists via localStorage.
- Command palette: press Cmd+K / Ctrl+K.
- Edit content in `src/app/page.tsx` (PROFILE object).
