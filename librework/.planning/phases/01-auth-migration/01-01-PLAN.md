    - Create `verify_stack_auth_token(access_token: str) -> dict` — gets signing key from JWKS client, calls `jwt.decode()` with `algorithms=["ES256"]`, `audience=settings.STACK_PROJECT_ID`. Returns the decoded payload dict (contains `sub`, `email`, `name`, etc.).
    - Handle `jwt.ExpiredSignatureError`, `jwt.InvalidTokenError` — re-raise as appropriate exceptions that dependencies.py can catch

    REWRITE `librework/backend/app/core/dependencies.py`:
    - Remove ALL existing imports of jose, passlib, security module
    - Import `verify_stack_auth_token` from `app.core.stack_auth`
    - Import `get_supabase` from `app.core.supabase`
    - Import `HTTPException`, `Request`, `Depends` from fastapi
    - Import `UserResponse` from `app.schemas`

    `async def get_current_user(request: Request) -> UserResponse`:
      1. Extract token from `request.headers.get("x-stack-access-token")` — if missing, raise 401
      2. Call `verify_stack_auth_token(token)` — wrap in try/except, raise 401 on failure
      3. Extract `stack_auth_id = payload.get("sub")` — if missing, raise 401
      4. Query Supabase: `supabase.table("users").select("*").eq("stack_auth_id", stack_auth_id).single().execute()`
      5. If no data, raise 401 "User not found in database"
      6. Return `UserResponse(**response.data)`

    `async def get_current_owner(request: Request) -> UserResponse`:
      1. Call `get_current_user(request)` to get user
      2. Check `user.role in ("owner", "admin")` — if not, raise 403
      3. Return user

    `async def get_current_admin(request: Request) -> UserResponse`:
      1. Call `get_current_user(request)` to get user
      2. Check `user.role == "admin"` — if not, raise 403
      3. Return user

    Keep the same function signatures that existing routers expect (they use `Depends(get_current_user)` and expect `UserResponse`). Check the existing `UserResponse` schema to ensure it has a `role` field — if not, it needs to be verified from the schemas.
