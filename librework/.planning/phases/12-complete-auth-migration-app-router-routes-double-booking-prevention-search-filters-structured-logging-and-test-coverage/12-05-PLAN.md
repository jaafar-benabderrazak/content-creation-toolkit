       **`mock_supabase_client` fixture** (autouse=True): Patches `app.core.supabase.get_supabase` to return a `MagicMock()`. Yields the mock for test assertions.

       **`owner_headers` fixture**: Same as `authenticated_headers` but returns user with `role: "owner"`. Yields headers dict.

       **`admin_headers` fixture**: Same but `role: "admin"`. Yields headers dict.

       The key challenge: `dependencies.py` calls `verify_stack_auth_token(token)` then looks up the user in Supabase by `stack_auth_id`. Both must be mocked. Read `dependencies.py` to understand the exact call chain and mock accordingly.
